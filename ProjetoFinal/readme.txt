Projeto final de Sistemas Reativos.

Utilizei o módulo de simulação física 2d do próprio love para desenvolver um simples jogo basquete.
Os jogadores se utilizam de um nodeMcu para jogar, onde o botão 1 é usado para arremesar, e o 2 para "aceitar" seu turno.
Os arquivos de cada mcu precisam ser atualizados para conectar a internet primeiro.
